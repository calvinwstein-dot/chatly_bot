📋 Setup Guide

End-to-end workflow from creation to subscription

🔵 Phase 1: Business Profile Creation

  • Location: server/businessProfiles/
  • File Format: ClientName.json
  • Required: businessName, displayName, industry, description, services, botPersonality, contactInfo, workingHours
  • ⚠️ businessName must match filename exactly

🟣 Phase 2: Stripe Configuration

  • Deploy server → Get public URL
  • Stripe Dashboard → Webhooks → Add endpoint
  • URL: https://your-url/stripe/webhook
  • Events: checkout.session.completed, customer.subscription.updated/deleted
  • Add webhook secret to .env

🟠 Phase 3: Client Journey

1️⃣ Consultation → 2️⃣ Setup Fee ($169) → 3️⃣ Profile Creation → 4️⃣ Admin Config → 5️⃣ Widget Install → 6️⃣ Subscription → 7️⃣ Go Live!

🔴 Quick Reference

  • Admin: http://localhost:3001/admin
  • Widget Test: http://localhost:3001/widget/?business=Name
  • Stripe: dashboard.stripe.com
