# Logo Files

Place your business logo images in this folder.

## Usage:
1. Add your logo file (e.g., `henri-logo.png`)
2. Update the business profile JSON with:
   ```json
   "logoUrl": "/public/logos/henri-logo.png"
   ```

## Supported formats:
- PNG (recommended for transparency)
- SVG (best for scalability)
- JPG

## Recommended size:
- Width: 150-200px
- Keep file size under 100KB for fast loading
